# Momentum Wellness Launch Messages

Use these drafts for the first launch period. Personalize before sending.
Avoid guarantees, medical claims, healthcare-cost savings promises, fake
results, and pressure-based language.

## Launch Positioning

Momentum Wellness is an app-based fitness, nutrition, habit, accountability,
and community wellness company for individuals and companies.

Primary launch message:

Momentum Wellness helps people build better health habits through structure,
accountability, and a modern app-based coaching system.

## Social Media Announcement

This is an app-based wellness coaching system built around training,

For individuals, Momentum Wellness provides structure, progress tracking, and
support through the app.
