# Momentum Wellness Onboarding Documents

This folder contains draft onboarding content for the Adult Strength program.

- `ADULT_STRENGTH_ONBOARDING_CONTENT.md` contains the welcome email, welcome
  video script, first-sign-in message, consultation questions, review rule, and
  Trainerize setup locations.
- `OPERATING_PROCEDURES.md` contains the staff procedures for individual
  purchases, failed payments, cancellations, manual client entry, and corporate
  wellness inquiries.
- `Momentum_Wellness_Liability_Waiver_TEMPLATE.docx` is an editable liability
  waiver template. It must be reviewed by a New Mexico attorney before use.
- `Momentum_Wellness_Liability_Waiver_TEMPLATE.pdf` is the matching printable
  review copy.
- `Momentum_Wellness_Exercise_Readiness_Consultation_Form.docx` is a printable
  custom readiness and consultation form. It is not the official PAR-Q+.
- `Momentum_Wellness_Exercise_Readiness_Consultation_Form.pdf` is the matching
  printable copy.

Do not store completed client forms, signatures, medical details, or other
private client information in this repository.
